Repositório para armazenar templates da Alura para usar com automações do n8n

# Primeira vez

# 1. Inicializar Git no projeto
git init

# 2. Adicionar todos os arquivos
git add .

# 3. Fazer o primeiro commit
git commit -m "Primeiro commit - Template DOCX"

# 4. Renomear branch para main (padrão atual do GitHub)
git branch -M main

# 5. Conectar com o GitHub (use a URL que você copiou)
git remote add origin https://github.com/SEU-USUARIO/SEU-REPO.git

# 6. Enviar para o GitHub
git push -u origin main

# Modificações

git status

git add .
git commit -m "Atualiza template com novos campos"
git push
